# API Katalog

A public API catalog where users can browse, search, and review APIs by category.

## Tech Stack

- **Framework:** Next.js 16 / TypeScript
- **Database & Auth:** Supabase
- **Styling:** Tailwind CSS

## Features

- Browse APIs by category
- Search and filter
- User reviews & star ratings
- Admin panel for managing API entries
- Auth via Supabase

## Setup

```bash
npm install
cp .env.example .env.local  # add your Supabase keys
npm run dev
```
